# 🌐 Social Links Profile

Uma página de perfil social minimalista que centraliza links pessoais e profissionais em um único card responsivo.
O projeto prioriza boas práticas de HTML semântico, CSS moderno e acessibilidade tipográfica.

---

## 🚀 Demonstração

A interface apresenta:

* Foto de perfil
* Nome e localização
* Descrição curta
* Botões com links sociais

Layout limpo, centralizado e adaptável a diferentes telas.

---

## 🛠️ Tecnologias utilizadas

* HTML5 semântico
* CSS3 moderno (Flexbox / Grid)
* Variáveis CSS
* Tipografia responsiva (`rem`, `clamp`)
* Boas práticas de acessibilidade
  
---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

* Estruturação semântica de páginas simples
* Centralização de layouts com CSS moderno
* Uso profissional de variáveis CSS
* Tipografia fluida e acessível
* Construção de UI limpa e consistente

---

## 📱 Responsividade

Projeto desenvolvido com abordagem **mobile-first**, garantindo:

* Leitura confortável em telas pequenas
* Escala tipográfica fluida

---

## ♿ Acessibilidade

* Uso de unidades relativas (`rem`)
* HTML semântico (`main`, `article`, etc.)
* Contraste adequado
* Estados de foco visíveis
